# AssetFlow Screens

## Screen 1 - Home

The Home screen provides access to the two main workflows.

Buttons:

- Import Shipment
- Assign Device
- Settings

---

## Screen 2 - Import Shipment

Purpose:

Import a shipment of new devices into Freshservice.

Features:

- Upload invoice
- Preview imported devices
- Show duplicate devices
- Create new assets

---

## Screen 3 - Assign Device

Purpose:

Assign an existing Stock asset to a user.

Features:

- Search by serial number
- Search by display name
- Scan barcode
- Scan serial number using phone camera (OCR)
- Manual serial number entry

Updates:

- Display Name
- Used By
- Department
- Asset State

---

## Screen 4 - Settings

Purpose:

Manage AssetFlow configuration.

Items:

- Device catalog
- Default values
- Freshservice connection settings
